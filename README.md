# La vie d'un étudiant : Kinshasa & USA

Un jeu textuel interactif développé en **Python** qui retrace le parcours semé d'embûches et de succès d'un étudiant de l'ISTA à Kinshasa jusqu'à son aventure universitaire et ses premiers stages aux États-Unis.

## 🚀 Fonctionnalités
- **Saison 1** : Gestion du quotidien à Kinshasa (transports, contrôles de reçus à l'ISTA, préparation du Visa avec Sarah).
- **Saison 2** : L'arrivée aux USA, participation à un Hackathon et entretien de stage technique.
- **Saison 3** : L'Inbound du succès et la création d'un centre de programmation pour aider la communauté de Kinshasa.

## 🛠️ Technologies
- Python 3
- GitHub pour le versioning

## 🎮 Comment jouer ?
Copiez le code du fichier `Jeu.py`, lancez-le dans votre terminal ou votre application de code favorite, et faites vos choix !
